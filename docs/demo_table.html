Table Helper 演示 - 员工数据表

掌控每一个网页表格 - 精确选择与智能处理

使用说明

  • 单击单元格：选择单个单元格（按住Ctrl/Cmd可多选）
  • 按住Alt + 点击：选择整列
  • 按住Ctrl + Alt + 点击：选择整行
  • 按住Ctrl、Shift + 点击：选择范围（从选中单元格到当前点击）
  • 按住Ctrl + 双击单元格：选择整个表格
  • 选择数据后：底部会显示统计面板和SQL生成器
  • 按 Ctrl+C：复制选中的表格数据

功能特性

精确选择

像Excel一样在网页表格中进行精确选择，支持单元格、行列、区域等多种选择模式

智能统计

自动计算选中数据的统计信息，包括总数、平均值、最值等

SQL生成

一键生成标准SQL语句，便于数据导入数据库或进一步处理
